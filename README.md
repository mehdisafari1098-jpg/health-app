# سیستم سلامت من 🩺

یک وب‌اپ شخصی فارسی (PWA) برای ثبت روزانه عادت‌ها، وزن و خواب — با نمودار رشد.
قابل نصب روی iPhone به‌صورت اپ مستقل.

## اجرا روی کامپیوتر خودت
1. Node.js را از nodejs.org نصب کن (نسخه LTS)
2. در پوشه پروژه، ترمینال را باز کن و بزن:
   npm install
   npm run dev
3. آدرسی که نشان می‌دهد (معمولاً http://localhost:5173) را در مرورگر باز کن

## دیپلوی روی Vercel (رایگان)
1. پروژه را روی GitHub بگذار
2. در vercel.com با GitHub وارد شو → Add New → Project → ریپو را انتخاب کن → Deploy
3. Vercel خودش همه‌چیز را تشخیص می‌دهد؛ چیزی را تغییر نده

## نصب روی iPhone
1. لینک Vercel را در **Safari** باز کن (نه کروم، نه داخل اپ دیگر)
2. دکمه Share (مربع با فلش) → **Add to Home Screen**
3. از این به بعد آیکون سبز روی صفحه اصلی، اپ را مستقل و تمام‌صفحه باز می‌کند

## نکته مهم درباره داده‌ها
داده‌ها در حافظه همان مرورگر/اپ ذخیره می‌شوند (localStorage).
اگر اپ را از صفحه اصلی حذف کنی یا داده‌های Safari را پاک کنی، داده‌ها هم پاک می‌شوند.
